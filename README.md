# T2AutoTron Home Assistant Add-ons Repository

This repository contains Home Assistant add-ons maintained by gregtee2.

## Add-ons

### [T2AutoTron](./t2autotron)

![Supports amd64 Architecture][amd64-shield]
![Supports aarch64 Architecture][aarch64-shield]
![Supports armv7 Architecture][armv7-shield]

Visual node-based smart home automation editor with real-time device control.

## Installation

1. Add this repository to your Home Assistant instance:

   [![Add repository to my Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fgregtee2%2Fhome-assistant-addons)

   Or manually:
   - Go to **Settings** → **Add-ons** → **Add-on Store**
   - Click the **⋮** menu (top right) → **Repositories**
   - Add: `https://github.com/gregtee2/home-assistant-addons`

2. Find your desired add-on in the store and install it.
